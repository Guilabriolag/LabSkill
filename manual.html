📖 Skill Builder · Manual do Usuário

Crie, teste e exporte skills para IA — guia prático e interativo

✨ O que é o Skill Builder?

Skill Builder é uma ferramenta que permite criar instruções detalhadas (skills) para assistentes de IA como o Claude. Você descreve o que quer que a IA faça, define gatilhos, regras e obtém um arquivo .skill (ou .md) pronto para usar.

💡 Dica: Uma skill é como um "superpoder" — um conjunto de instruções, exemplos e limites que moldam a resposta da IA.

🎯 Fluxo básico

1. Escolha o modo POP (rápido) ou PRO (detalhado)
2. Preencha o que a IA deve fazer
3. Defina triggers (frases que ativam a skill)
4. Ajuste regras, stack e exemplos
5. Teste com o simulador
6. Baixe .skill ou .md

🛠️ Modo Pro — controle total

No modo profissional você gerencia: nome, versão, descrição, stack tecnológica, regras obrigatórias, anti-erros, exemplos de ativação/não ativação e muito mais.

🔧 Campos essenciais

  • Nome da skill → identificador único (ex: lqr-menu-builder)
  • Descrição / trigger principal → o Claude lê esse texto para decidir se ativa a skill
  • O que ela faz → comportamento detalhado e formato de saída
  • Stack → tags como Vanilla JS, Cloudflare, PWA, Bitcoin
  • Regras → princípios obrigatórios que a IA deve seguir
  • Anti-erros → comportamentos proibidos explicitamente

⚙️ Como funciona o teste?

O sistema compara a frase com triggers cadastrados e com o texto da descrição principal. Se houver correspondência, o resultado é "Skill ATIVARIA".

📦 Exportar — .skill vs .md

Após criar sua skill, você pode baixá-la de duas formas:

  • .skill → arquivo compactado (.zip disfarçado) contendo SKILL.md, manifest.json e um icon.svg personalizado. Pronto para distribuição.
  • .md → apenas o arquivo Markdown com toda a estrutura da skill (fácil de ler e editar manualmente).
📌 Copiar markdown: útil para colar diretamente em chats, documentação ou repositórios.

📁 Estrutura interna do .skill

minha-skill.skill (ZIP)
 ├─ SKILL.md          → instruções completas (frontmatter + markdown)
 ├─ manifest.json     → metadados (id, versão, autor, capacidades)
 └─ icon.svg          → ícone gerado com a primeira letra da skill
